# Push_Swap

Program that sorts a linked list based on the argument entered in the parameter, Using the Turkish sort.

## Installation

```bash
git clone https://github.com/Nerada00/push_swap.git
cd push_swap
make
```

## Utilisation

After compiling enter in arg 
```bash
Only int value;
Value can be > 0 or < 0
```
